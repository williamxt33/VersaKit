Group Generator — About & How to Use

About this Tool

Quickly and fairly split people into groups for classrooms, teams, workshops, events, and meetings. This tool supports dividing by number of groups or by desired people-per-group, and uses a random shuffle to ensure unbiased distribution.

Results can be copied or downloaded for easy sharing and recordkeeping.

How to Use

  1. Enter names: Paste or type one name per line into the names list.
  2. Choose division method: Select whether to split by number of groups or by people per group.
  3. Configure options: Adjust sorting, group size, or any balancing options available.
  4. Generate: Click "Generate Groups" to create groups instantly.
  5. Export: Use "Copy All Groups" or "Download as Text" to save results for distribution.

Best Practices

  • Use clear, consistent name formats for easier review.
  • Regenerate to try different combinations if you need alternate groupings.
  • Save or export results before making changes so you can reproduce setups.
  • For sensitive contexts, anonymize participant names before sharing results publicly.

Use Cases & Examples

Classrooms: Create project teams quickly.

Events: Organize attendees into discussion or breakout groups.

Sports/Activities: Make balanced teams for casual play.
