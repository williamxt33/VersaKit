Barcode Format Guide

Code 128

Character Set: Full ASCII (128 characters)

Length: Variable

Best For: Universal applications, logistics, inventory

Example: ABC123456

EAN-13

Character Set: Numeric only

Length: 13 digits (12 + check digit)

Best For: Retail products worldwide

Example: 5901234123457

UPC-A

Character Set: Numeric only

Length: 12 digits (11 + check digit)

Best For: US and Canada retail products

Example: 012345678905

Code 39

Character Set: Alphanumeric + special chars

Length: Variable

Best For: Non-retail, automotive, defense

Example: ABC-123

ITF-14

Character Set: Numeric only

Length: 14 digits

Best For: Packaging and shipping containers

Example: 12345678901231
