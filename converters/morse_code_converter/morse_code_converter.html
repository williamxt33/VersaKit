Morse Code Reference

Morse Code Timing:

• Dot (.) = 1 unit
• Dash (-) = 3 units
• Space between parts of letter = 1 unit
• Space between letters = 3 units
• Space between words = 7 units

Common Prosigns:

• Start of message (KA): -.-.-
• End of message (AR): .-.-.
• Error (HH): ........
• Wait (AS): .-...
• Understood (SN): ... -.
